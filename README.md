# Program Description

`fourier.py` is the main program, with specific Fourier series function definitions and settings under the `func` directory.

The import statement in `fourier.py` allows you to select which function to draw.

I believe you can create and add your own settings for the series, so if you like, you can fork it and experiment with different functions.
